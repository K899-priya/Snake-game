# 🐍 Snake Game Pro

A modern snake game built using Python and Pygame with UI menu, sound effects, and high score tracking.

## 🚀 Features

- Smooth gameplay
- Score tracking
- High score saving
- Pause feature
- Menu screen
- Sound effects
- Increasing difficulty

## 🛠 Tech Stack

Python, Pygame

## ▶️ Installation

pip install -r requirements.txt  
python snake.py

## 🎮 Controls

Arrow keys → Move  
P → Pause  
R → Restart
